CSS 动画 (Animations) 允许元素从一种样式逐渐变化为另一种样式。与 transition 不同，animation 可以控制多个步骤、循环播放以及更复杂的运动曲线。

1. 基础动画 (@keyframes)

使用 @keyframes 定义动画序列，然后通过 animation 属性应用它。

Slide
@keyframes slide-right { from { transform: translateX(-50px); } to { transform: translateX(50px); } } .box { /* 动画名称 时长 循环次数 方向 */ animation: slide-right 2s infinite alternate; }

2. 多阶段关键帧

使用百分比来精确控制动画的中间状态。

Pulse
@keyframes color-change { 0% { background: #7b6cff; transform: scale(1); } 50% { background: #ff6c6c; transform: scale(1.2); } 100% { background: #7b6cff; transform: scale(1); } }

3. 加载动画 (Infinite Loop)

结合 rotate 和 linear 线性时间函数制作加载圈。

@keyframes spin { to { transform: rotate(360deg); } } .loader { border: 4px solid #ddd; border-top-color: #7b6cff; border-radius: 50%; /* 线性速度，无限循环 */ animation: spin 1s linear infinite; }

4. 缓动函数 (Timing Function)

使用 cubic-bezier 可以制作更有物理质感的弹跳效果。

Bounce
@keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-40px); } } .ball { animation: bounce 1s cubic-bezier(0.28, 0.84, 0.42, 1) infinite; }

常用属性速查

  • animation-name: @keyframes 的名称
  • animation-duration: 动画完成一个周期所需时间 (如 2s)
  • animation-timing-function: 速度曲线 (linear, ease, ease-in-out, cubic-bezier)
  • animation-delay: 动画开始前的延迟
  • animation-iteration-count: 播放次数 (数字 或 infinite)
  • animation-direction: 播放方向 (normal, reverse, alternate)
  • animation-fill-mode: 动画结束后元素的样式状态 (forwards, backwards, both)
