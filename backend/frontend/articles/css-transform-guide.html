CSS transform 属性允许你旋转，缩放，倾斜或平移给定元素。这是现代 Web 动画和布局的核心属性之一。本文将通过可视化示例介绍各个参数的作用。

1. Translate (平移)

translate() 方法从其当前位置移动元素（根据 X 轴和 Y 轴参数）。它不会影响文档流中的其他元素。

X+20px
translateX(20px)
Y-10px
translateY(-10px)
X+20, Y+20
translate(20px, 20px)

2. Rotate (旋转)

rotate() 方法在一个给定的度数内旋转元素。正值是顺时针旋转，负值是逆时针旋转。

45deg
rotate(45deg)
-90deg
rotate(-90deg)
180deg
rotate(180deg)

3. Scale (缩放)

scale() 方法增加或减少元素的大小（根据给定的宽度和高度参数）。值大于 1 为放大，0-1 之间为缩小。

0.8
scale(0.8)
1.2
scale(1.2)
ScaleX 1.5
scaleX(1.5)

4. Skew (倾斜)

skew() 方法包含两个参数值，分别表示 X 轴和 Y 轴倾斜的角度。

SkewX 20deg
skewX(20deg)
SkewY 10deg
skewY(10deg)
Skew 15,10
skew(15deg, 10deg)

5. 组合变换与顺序

你可以组合多个变换，顺序很重要（从左到右应用）。坐标系会随着变换而改变。

Move -> Rotate
translateX(50px) rotate(45deg)
Rotate -> Move
rotate(45deg) translateX(50px)

注意上面两个例子的区别：先旋转再平移时，平移的方向是相对于旋转后的坐标系的（即沿着 45 度角方向移动）。

6. Transform Origin (变换原点)

transform-origin 允许你改变被转换元素的位置。默认原点是中心点 (50% 50%)。

Top Left
origin: top left
Center
origin: center
Bottom Right
origin: bottom right
